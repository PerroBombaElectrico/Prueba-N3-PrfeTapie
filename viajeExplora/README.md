# ViajeXplora

Landing Page desarrollada con React y Bootstrap.

## Tecnologías utilizadas

- React
- Bootstrap 5
- Vite

## Instalación

```bash
npm install
```

## Ejecutar

```bash
npm run dev
```

## Funcionalidades

- Navbar responsive con búsqueda
- Hero principal con CTA
- Carousel de destinos
- Modal de detalle de paquete
- Acordeón de itinerario
- Tarjetas de destinos
- Footer responsive
